Nesting Storyboards

Storyboards can be nested to create more modular screen flows. To add a nested storyboard use one of these methods.

  • Drag & Drop storyboard files directly from the Project Explorer view into the editor. You may have to uncheck the Link with Editor option () in Project Explorer view first.
  • Copy & Paste storyboard files from the Project Explorer view into the editor.
